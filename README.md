## Tech Stack
* Next.js
* Tailwind CSS
* Redux Toolkit
* Sanity
* React Hook Form
* Stripe
* Vercel


## Deployment

To deploy this project run

```bash
  npm run deploy
```

For installing dependencies
```bash
  npm install --legacy-peer-deps
```
